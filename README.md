# jellyfish-rhopalium
single-cell analyses of jellyfish rhopalium repair

## Data Availability
Processed single-cell RNA sequencing (scRNA-seq) data generated in this study are deposited in Zenodo:
https://zenodo.org/doi/10.5281/zenodo.19435350

## Reproducibility
All analyses in this study can be fully reproduced using the provided code and processed data. The analysis pipeline follows standard single-cell RNA-seq workflows, with detailed parameter settings documented in the R script.
